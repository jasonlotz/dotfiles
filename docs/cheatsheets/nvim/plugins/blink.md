# Blink

## Auto completion (in insert mode)

- `<ctrl> + p`: Previous suggestion
- `<ctrl> + n`: Next suggestion
- `<ctrl> + <space>`: Show completions
- `<ctrl> + e`: Close
- `<ctrl> + y`: Confirm completion

Note: See :help ins-completion for more information.
