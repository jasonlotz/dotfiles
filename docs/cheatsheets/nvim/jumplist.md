# Jumplist

## Navigate jumplist

- `:jumps`: Show jumplist

- `<C-^>`: Jump to previous buffer
- `<C-i>`: Jump to newer position in jumplist
- `<C-o>`: Jump to older position in jumplist

- `''`: Jump to previous line in current buffer
- ``: Jump to previous position (line and column) in current buffer

## Fuzzy find

- `<leader>fj`: Fuzzy find jumplist
