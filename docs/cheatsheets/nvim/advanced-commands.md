# Advanced Commands

## Useful g commands

- `ga`: Show ASCII value of character under cursor
- `gv`: Reselect last visual selection
- `g;`: Go to last change
- `g,`: Go to previous change
- `gi`: Go to last insert position
- `g*`: Search for word under cursor (forward)
- `g#`: Search for word under cursor (backward)

## Goofy g commands

- `g??`: Encrypt/decrypt current line with ROT13
- `g?{motion}`: Encrypt/decrypt text covered by {motion} with ROT13
